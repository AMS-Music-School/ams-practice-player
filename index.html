【Audio File Mode】操作ガイド
  • ファイル選択: デバイス内の音楽ファイルを選択してください。BPMと無音部分を自動解析します。
  • BPM連動: BPM数値を手入力すると速度が変わります。速度を変えるとBPMが計算されます。
  • ループ設定: 波形をドラッグして範囲を選択すると、その区間が繰り返し再生されます。
  • ピッチ調整: スピードを変えずに半音単位（±6）で変更可能です。
【YouTube Mode】操作ガイド
  • 動画の読み込み: YouTubeのURLを貼り付けて「LOAD」ボタンを押してください。
  • ループ区間の指定: 動画を再生し、開始したい位置で「SET START」、終了位置で「SET END」を押します。
  • 微調整: 「-1s / +1s」ボタンで、ループの開始・終了位置を1秒単位で調整できます。
  • ループ実行: 「LOOP OFF」ボタンを押して「LOOP ON」に切り替えると、指定区間が繰り返されます。
